How to read this data

What is a CIP code?

CIP stands for Classification of Instructional Programs — a taxonomy maintained by NCES (National Center for Education Statistics) to standardize how educational programs are categorized. The 6-digit code (e.g. {{ prog.cip }}) identifies the specific program type. The first two digits identify the broad field of study.

How are programs linked to occupations?

The occupation links are derived from the NCES CIP 2020 → SOC 2018 crosswalk. Each program's CIP code is matched to one or more SOC codes (Standard Occupational Classification). This is a structural mapping, not a guaranteed employment outcome. Confidence values reflect the directness of the mapping (1.0 = direct crosswalk match).

What are "completions"?

Completions are awards conferred during the academic year — degrees, certificates, and other credentials actually earned by students. This is not the same as enrollment or graduation rates. A single student can earn multiple awards in one year.

Why does this program show — instead of a completion count?

NCES suppresses completion values when fewer than 3 students completed a program in a given year, to protect student privacy. A dash (—) means data exists but is withheld. It does not mean zero completions.

Data currency

Program and completion data is sourced from IPEDS Completions (C) 2023–24 (data year 2024). IPEDS data is collected annually; this reflects the most recent finalized release available. CIP→SOC crosswalk is NCES CIP 2020 / SOC 2018.
